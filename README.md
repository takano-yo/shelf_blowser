# shelf_blowser

CiNii Books API で取得した学術書の書誌データを **本棚のように表示** し、
図書館の書架で行うようなブラウジング（請求記号順に棚を眺め、隣接する関連書に
偶然出会う体験）を Web で再現する学術検索支援アプリ。

試作版として **日本近代文学** 分野を対象とする。

---

## コンセプト

CiNii の通常検索では失われる「棚を眺めて歩く」ブラウジング体験を Web で再現する。

- **棚の並び順**: `cinii:ownerCount`（所蔵館数）の降順。所蔵が多い＝広く読まれている
  主要書ほど棚の手前（先頭）に並ぶ。
- **検索 = 棚へのワープ**: 著者・出版者・出版年などのファセットで絞り込み、
  該当する棚の位置へ移動する。

---

## データソース

`source/日本近代文学.json`

CiNii Books OpenSearch API のレスポンス（クエリ: `日本 AND 近代 AND 文学`,
`count=10000`, `sortorder=5`）。ヒットした **5,212 件** の書誌が `@graph[0].items` に
格納されている。各 item は一覧レベルの書誌（タイトル・著者文字列・出版者・年・
所蔵館数・シリーズ・ISBN）を持ち、`rdfs:seeAlso` に詳細レコード（`.json`）への
リンクを持つ。

### データの役割分担（重要）

- **トップページの本棚表示** … OpenSearch のレスポンス（＝`source` の一覧情報）
  **だけで充足する**。詳細 API は不要。
- **件名・著者名を含む詳細検索** … 図書一件ごとの詳細レコード（`.json`）にある
  件名（`foaf:topic`）・典拠著者（`foaf:maker`）・分類（`dc:subject`）を利用する。
  これらは OpenSearch には含まれないため、`fetch` モジュール（現時点では未定義）が
  詳細検索を有効化する段階で取得する。

## アーキテクチャ（3 モジュール構成のパイプライン）

```
source/日本近代文学.json  (CiNii OpenSearch)
        │
        │  ＝トップページ本棚表示はこれだけで充足
        ▼
┌─────────────┐   検索項目の抽出・正規化・データ加工
│   build/    │   → site/data/*.json（ownerCount 降順で整列）
└─────────────┘
        │
        ▼
┌─────────────┐   静的サイト（GitHub Pages 公開）
│   site/     │   本棚ビュー + 基本検索。将来のモバイル対応を見据える
└─────────────┘
        ▲
        │  ＝件名・著者名の「詳細検索」を有効化する段階で合流
┌─────────────┐   図書一件ごとの詳細レコード（件名・典拠著者・分類）を取得
│   fetch/    │   ※ 現時点では未定義（詳細検索専用。トップ表示には不要）
└─────────────┘
```

各モジュールの詳細方針は各フォルダ内の `README.md` を参照。

| モジュール | フォルダ | 役割 | 現状 |
|---|---|---|---|
| ① API 取得 | [`fetch/`](fetch/README.md) | **詳細検索用**に図書一件ごとの詳細書誌（件名・著者名・分類）を取得 | **未定義**（方針のみ） |
| ② データ加工 | [`build/`](build/README.md) | OpenSearch から検索項目抽出・整形・索引生成 | 現状 JSON から構築 |
| ③ サイト実装 | [`site/`](site/README.md) | 本棚 UI・基本検索（＋将来の詳細検索） | 現状 JSON から構築 |

---

## 確定済みの要件

1. **棚の並び順** = `cinii:ownerCount` の降順
2. **トップページ表示** = `source/日本近代文学.json`（OpenSearch）だけで充足。
   詳細 API は不要
3. **詳細検索（件名・著者名）** = 図書一件ごとの詳細 API 情報を利用。
   この取得を担う `fetch` モジュールは現時点では未定義
4. **公開先** = GitHub Pages。ただし**将来的にモバイル（スマホ）表示にも
   対応できる拡張性**を持たせる
