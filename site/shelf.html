データについて

書誌データ

書誌データは国立情報学研究所（NII）が提供する CiNii Books OpenSearch API から取得し、CC BY 4.0 ライセンスのもとで使用しています。

書影データ

書影（表紙画像）は openBD API から取得し、書籍の紹介を目的として使用しています。

デフォルトの並び順について

「すべて」タブのみ独自の重みづけによる並び順を採用しており、それ以外のタブ（単著/共著・編集書・シリーズ）は所蔵館数の多い順に並んでいます。

「すべて」タブの重みづけは次のとおりです。

  • シリーズはまとめたうえで、平均6冊に1冊の割合で配置する
  • 編集書も同様に、平均6冊に1冊の割合で配置する
  • 残り（平均6冊中4冊）は単著・共著が占める
  • 単著・共著を使い切った後は、残りのシリーズ・編集書を並べる
  • いずれの場合も、所蔵館数が多い順に並べる

データ出典: CiNii Books OpenSearch API／表紙画像: openBD
